# Proyecto1_TBDD_POO

Este proyecto es un sistema simple hecho en Python para registrar deportistas y competencias.

## Descripción

El programa permite crear distintos tipos de deportistas, como futbolistas, tenistas y atletas. También permite registrarlos, ordenarlos por puntaje, inscribirlos en competencias y guardar sus resultados.

## Clases usadas

- `Deportista`: clase base abstracta.
- `Futbolista`: clase hija para deportistas de fútbol.
- `Tenista`: clase hija para deportistas de tenis.
- `Atleta`: clase hija para deportistas de atletismo.
- `Registro`: guarda la lista de deportistas.
- `Competencia`: permite inscribir participantes y registrar resultados.

## Cómo ejecutar

En la terminal, usar:

```bash
python3 main.py
